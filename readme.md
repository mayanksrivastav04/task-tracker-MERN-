# MERN Task Tracker

A responsive Task Tracker web application built using the MERN Stack.

## 🚀 Features

- Create, Read, Update & Delete Tasks (CRUD)
- Mark Tasks as Completed
- Task Priority (Low, Medium, High)
- Form Validation
- REST API Integration
- Responsive UI
- MongoDB Database

## 🛠 Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- Tailwind CSS
- Axios

## 📂 Run Locally

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
